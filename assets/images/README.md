# Imágenes del álbum

Coloca aquí las fotos con **exactamente** estos nombres (formato `.jpg`). También puedes usar `.png` o `.webp` cambiando la extensión en `assets/js/config.js`.

| Archivo            | Uso                          |
|--------------------|------------------------------|
| `cover-mockup.jpg` | Portada del libro            |
| `back-cover.jpg`    | Contraportada                |
| `foto-01.jpg`      | Página 1 (imagen izquierda)  |
| `foto-02.jpg`      | Página 2                     |
| `foto-03.jpg`      | Página 3                     |
| `foto-04.jpg`      | Página 4                     |
| `foto-05.jpg`      | Página 5                     |
| `foto-06.jpg`      | Página 6                     |
| `foto-07.jpg`      | Página 7                     |

Para cambiar títulos o mensajes, edita `assets/js/config.js`.
